# Home Assistant Add-on: zhonghong_vrf

## 关于

[zhonghong_vrf](https://github.com/xswxm/zhonghong_vrf/blob/master/README.md)是一个基于HTTP控制中弘空调的应用。


## 配置

#### Option `broker`

mqtt服务器

#### Option `port`

mqtt服务器端口

#### Option `username`

mqtt服务器用户名

#### Option `password`

mqtt服务器密码

#### Option `gateway`

中弘网关IP地址
